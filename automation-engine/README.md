# Automation Engine

The CodeBattle automation engine provides n8n-like workflow automation capabilities.

## Features

- Visual workflow builder
- Pre-built nodes for common tasks
- Custom node development
- Event-driven triggers
- Workflow scheduling

## Node Types

### API Nodes
- HTTP Request
- REST API
- GraphQL API
- Webhook

### Database Nodes
- PostgreSQL Query
- Redis Operation
- MongoDB Query

### Control Flow
- Condition
- Switch
- Loop
- Merge

### Data Transformation
- Map
- Filter
- Transform
- Join

### Code Execution
- JavaScript
- Python
- Shell Script

## Usage

Create workflows using the visual builder in the frontend application.
